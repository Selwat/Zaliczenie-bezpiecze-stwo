        function szyfrujTekst(tekst, przesuniecie) {
          var zaszyfrowanyTekst = "";

          for (var i = 0; i < tekst.length; i++) {
            var znak = tekst[i];
    
            if (znak.match(/[a-z]/i)) {
              var kod = tekst.charCodeAt(i);
    
              if (kod >= 65 && kod <= 90) {
                znak = String.fromCharCode(((kod - 65 + przesuniecie) % 26) + 65);
              } else if (kod >= 97 && kod <= 122) {
                znak = String.fromCharCode(((kod - 97 + przesuniecie) % 26) + 97);
              }
            }
    
            zaszyfrowanyTekst += znak;
          }
    
          return zaszyfrowanyTekst;
        }

Sprawozdanie - Szyfr Cezara

Wprowadzenie
Celem tego projektu było stworzenie strony internetowej umożliwiającej szyfrowanie tekstu przy użyciu szyfru Cezara. Szyfr Cezara jest jednym z najprostszych szyfrów przesuwających, w którym każda litera tekstu jest zamieniana na inną literę o stałym przesunięciu. W ramach tego projektu wykorzystano języki HTML i JavaScript w celu implementacji strony oraz logiki szyfrowania.

Implementacja
Strona internetowa składa się z dwóch głównych komponentów: pliku HTML i skryptu JavaScript.

Plik HTML
Plik HTML stanowi strukturę strony internetowej. Zawiera formularz umożliwiający użytkownikowi wprowadzenie tekstu do zaszyfrowania oraz wybór przesunięcia. Na podstawie tych informacji użytkownik może zaszyfrować tekst. Plik HTML zawiera również element na stronie, w którym wyświetlany jest zaszyfrowany tekst.

Skrypt JavaScript
Skrypt JavaScript odpowiada za logikę szyfrowania tekstu. Implementuje funkcję szyfrujTekst, która przyjmuje tekst i przesunięcie jako argumenty. Wewnątrz funkcji iterujemy po każdym znaku tekstu. Jeśli znak jest literą alfabetu, obliczamy nowy kod znaku na podstawie przesunięcia i zamieniamy go na odpowiadającą literę. Funkcja zwraca zaszyfrowany tekst.
Dodatkowo, skrypt JavaScript zawiera funkcję zaszyfruj, która jest wywoływana po kliknięciu przycisku "Zaszyfruj" na stronie. Funkcja pobiera tekst i przesunięcie wprowadzone przez użytkownika, wywołuje funkcję szyfrujTekst i wyświetla zaszyfrowany tekst na stronie.

Podsumowanie
Dzięki temu projektowi udało się stworzyć stronę internetową umożliwiającą szyfrowanie tekstu przy użyciu szyfru Cezara. Użytkownik może wprowadzić tekst do zaszyfrowania i wybrać przesunięcie, a następnie zobaczyć zaszyfrowany tekst na stronie. Implementacja opiera się na prostym pliku HTML, który definiuje strukturę strony, oraz skrypcie JavaScript, który zawiera logikę szyfrowania. Należy jednak pamiętać, że szyfr Cezara jest bardzo prostym szyfrem i nie zapewnia wysokiego poziomu bezpieczeństwa.
